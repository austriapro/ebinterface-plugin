System - Voraussetzungen

Die Word Vorlage ist zur Verwendung in folgender Umgebung ausgelegt:

  • Microsoft Word 2007, Microsoft Word 2010, Microsoft Word 2013 mit allen Updates und Service Packs

  • .NET Framework 4.0 (wird ggf. mitinstalliert, benötigt Internetzugang)

  • Visual Studio Tools für Office 2010 (gilt für alle unterstützten Versionen von Word. wird ggf. mitinstalliert, benötigt Internetzugang)

  • Windows XP, VISTA, Windows 7 oder Windows 8 und 8.1

Aufgrund von Sicherheitseinstellungen in Word 2007 / 2010 / 2013 kann es erforderlich sein, die Installation von der lokalen Festplatte durchzuführen. Details dazu finden Sie im Vertrauensstellungscenter. (Office->Word-Optionen-Vertrauensstellungscenter->Einstellungen für das Vertrauensstellungscenter)

Hinweis: Für die Installation sind Administrator-Rechte erforderlich.
